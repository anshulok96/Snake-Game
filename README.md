# Snake-Game
This is a Snake Game created using html, js, and CSS. 

A modern, browser-based Snake game built with HTML, CSS, and vanilla JavaScript, featuring a neon/glassmorphism UI, smooth gameplay, keyboard + mobile controls, and a clean state-driven game loop.

## Live Gameplay

Classic Snake mechanics
Smooth canvas rendering
Increasing difficulty as you score
Game over modal with restart
